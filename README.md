# Data Cleaning & Preprocessing

## 📋 Descrição
Projeto de limpeza e pré-processamento de dados experimentais. Tratamento de 
valores faltantes, outliers e transformações necessárias para análise.

## 📊 Dados
- `logs_exp_us.csv` - Logs experimentais crus

## 🚀 Como Executar
```bash
pip install -r requirements.txt
jupyter notebook notebooks/analysis.ipynb
```

## 📈 Análises Realizadas
- Limpeza de dados
- Tratamento de valores faltantes
- Detecção e tratamento de outliers
- Transformação de variáveis

## 🛠️ Tecnologias
- Python 3.8+
- pandas, numpy, matplotlib, seaborn
